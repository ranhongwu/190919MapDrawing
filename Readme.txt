UniqueValueRenderer
2019/9/9

1、功能描述：
	实现要素唯一值符号化。

2、开发环境：
	操作系统：windows10
	编程语言：c#
	开发工具：vs2017、ArcEngine10.4、DevExpress插件
	平台：.net Framework4.6

3、解决方案中目录结构：
	|---README.txt			//说明文档
	|---FrmMapDrawing.cs		//主界面及代码
	|---FrmTitle.cs			//添加地图标题界面及代码
	|---FrmLegend.cs			//添加图例界面及代码
	|---FrmNorthArrow.cs		//添加指北针界面及代码
	|---FrmScaleBar.cs			//添加比例尺界面及代码
	|---FrmMapExport.cs		//地图导出图片界面及代码

4、注意：运行之前需要将ArcGIS安装目录下的Style文件夹下的“ESRI.style”文件拷入程序起始路径下的
Style文件夹下。
